# Word Letter Printer (Loop Combo)
This program keeps asking the user for words using a while‑loop and prints each letter on a new line using a for‑loop. The program stops when the user types "stop".

## Features
- Uses a while loop to repeatedly ask for words
- Stops immediately when the user types "stop"
- Uses a for loop to print each letter of the word
- Handles uppercase/lowercase by checking .lower()


## Learning Purpose
A small exercise to practice:
- combining while‑loops and for‑loops
- loop conditions and breaking out of loops
- iterating over strings
- user input handling
